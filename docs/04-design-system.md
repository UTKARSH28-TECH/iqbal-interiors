\# DESIGN SYSTEM



\# Purpose



This document defines the complete visual design system for the Iqbal Interiors website.



Every component, page, and section must follow these rules to maintain consistency and create a premium user experience.



\---



\# Design Philosophy



The website should feel like a luxury interior design studio.



Prioritize:



\* Simplicity

\* Large photography

\* Spacious layouts

\* Elegant typography

\* Premium materials

\* Subtle animations



Photography should always be the main visual element.



\---



\# Brand Personality



The interface should communicate:



\* Luxury

\* Professionalism

\* Modern design

\* Trust

\* Quality craftsmanship

\* Elegance

\* Warmth



Avoid loud or overly decorative UI elements.

Desktop



3–4 columns where appropriate.



Tablet



2 columns.

\---



\# Buttons



Primary



\* Gold background

\* White text

\* Rounded corners

\* Soft shadow

\* Smooth hover animation



Secondary



\* White background

\* Black border

\* Black text



Hover



\* Slight lift

\* Slight scale

\* 300ms transition



\---



\# Cards



Service cards



Gallery cards



Project cards



Should have:



\* Rounded corners

\* Soft shadow

\* Consistent spacing

\* High-quality image

\* Minimal text



\---



\# Border Radius



Buttons



14px



Cards



20px



Images



20px



Inputs



12px



Modals

\---



\# Images



Use real project photography.



Never use stock images.



Preserve originality.



Use Next.js Image component.



Optimize for fast loading.



Maintain aspect ratio.



\---



\# Hero Section



Requirements



\* Full-screen layout

\* Three-image slider

\* Dark overlay

\* Premium typography

\* Clear CTA buttons



Images should showcase real client work.



\---



\# Gallery



The gallery is the primary selling feature.



Requirements



\* Large images

\* Responsive grid

\* Lightbox

\* Category pages

\* Shareable URLs



Visitors should spend time browsing.



\---



\# Navigation



Sticky



Transparent over hero



Blur on scroll



Large mobile menu



Visible WhatsApp button



\---



\# Icons



Use Lucide React.



Prefer outline icons.



Avoid colorful illustrations.



\---



\# Animations



Use Framer Motion.



Allowed:



\* Fade

\* Slide

\* Image reveal

\* Gentle zoom

\* Hover lift



Duration



200–500ms



Animations should support the content rather than distract from it.



\---



\# Mobile Design



Design mobile-first.



Requirements



\* Large touch targets

\* One-column layouts

\* Comfortable spacing

\* Fast image loading

\* Easy navigation



\---



\# Accessibility



Follow WCAG guidelines.



Use:



\* Semantic HTML

\* Keyboard navigation

\* Visible focus states

\* Alt text

\* Accessible forms



\---



\# Things to Avoid



Do not use:



\* Bright gradients

\* Neon colors

\* Generic SaaS layouts

\* Glassmorphism

\* Heavy shadows

\* Cluttered sections

\* Tiny fonts

\* Auto-playing videos



The website should remain timeless and premium.



\---



\# Final Design Goal



Every page should look custom-designed for Iqbal Interiors.



The user should remember the photography, craftsmanship, and clean presentation—not the interface itself.



The design should support the client's work, never compete with it.

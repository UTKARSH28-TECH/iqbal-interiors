\# SITE ARCHITECTURE



\# Purpose



This document defines the complete information architecture of the Iqbal Interiors website.



Every page, section, route, and navigation item should follow this structure.



The architecture should support future growth without major restructuring.



\---



\# Website Navigation



Primary Navigation



\* Home

\* Gallery

\* Services

\* About

\* Contact



Navigation should remain simple and uncluttered.



\---



\# Homepage Structure



The homepage should tell the complete story of the business.



Order of sections:



1\. Navbar

2\. Hero Slider

3\. Trust Statistics

4\. Services

5\. Gallery Categories

6\. Featured Projects

7\. Why Choose Us

8\. Showroom Preview

9\. Owner Introduction

10\. Testimonials (future-ready)

11\. Contact CTA

12\. Footer



\---



\# Hero Section



Purpose



Create an immediate premium impression.



Features



\* Three-image hero slider

\* Real project photography

\* Large headline

\* Supporting description

\* Explore Gallery button

\* WhatsApp button



\---



\# Trust Statistics



Display animated counters.



Suggested metrics



\* Years of Experience

\* Completed Projects

\* Happy Clients

\* Product Categories



These values should be configurable.



\---



\# Services Section



Display the company's primary services.



Each service card contains:

\* TV Unit

\* Modular Kitchen

\* Lighting

\* Jhumars

\* Sanitary

\# Gallery Categories



This is one of the most important sections.



Each category links to its own dedicated gallery page.



Categories:



\* Bedroom

\* Hall

\* Kitchen

\* False Ceiling

\* TV Unit

\* Lighting

\* Jhumars

\* Sanitary

\* Office



Every category should display:



\* Cover image

\* Title

\* Image count (optional)

\* View Collection button



\---



\# Featured Projects



Display selected premium projects.



Each project should include:



\* Large cover image

\* Project title

\* Brief description

\* View Project button



Projects should alternate image alignment for visual variety.
